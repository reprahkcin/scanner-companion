# Quick Start Guide

Status: Current (supported)

## Installation

1. Clone: git clone https://github.com/reprahkcin/scanner-companion.git
2. Run: ./run_scanner.sh
3. Optional health check only: ./run_scanner.sh --doctor

## First Use

1. Test camera preview
2. Run calibration
3. Start capture
