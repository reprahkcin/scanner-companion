# Calibration Examples

This directory contains example calibration files for different lens configurations.

## Files

- `10mm calibration.json` - Example calibration for 10mm lens
- `14mm calibration.json` - Example calibration for 14mm lens  
- `sample_xmp_calibration.json` - Sample XMP calibration settings

These files can be used as templates or reference when setting up your own calibrations. Copy them to your working directory and modify the values as needed for your specific setup.
